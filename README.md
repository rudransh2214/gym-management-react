# 🏋️ Gym Membership Management System (React)

A modern, responsive, and feature-rich **Gym Membership Management System** built with **React.js**. This project was developed as a **Final Year College Project** to demonstrate modern frontend development practices, component-based architecture, and an intuitive user experience for gym management.

> **Note:** This is a frontend application built with React and Vite. All data is currently managed using dummy JSON data and Local Storage, making it ready for future backend integration.

---

## 📖 Project Overview

The Gym Membership Management System provides an interactive platform for both gym administrators and members. It simplifies gym operations by offering dashboards, membership management, trainer management, class booking, attendance tracking, workout plans, nutrition plans, payment management, and analytics.

The application is built with reusable React components and follows a scalable project structure for future enhancements.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/gym-membership-management.git
```

Navigate to the project

```bash
cd gym-membership-management
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
